Initiating Factory Reset

1. Enter ROMMON mode

To start, I'm going to carry out the following steps on the Cisco Catalyst 3550 switch to enter ROMMON mode:

  1. Unplug the power cable from the switch.
  2. Press and hold the MODE button on the front panel.
  3. While holding the MODE button, reconnect the power.
  4. Continue holding the button until you see the switch: prompt appear in your console session (PuTTY).

This procedure interrupts the normal boot process and forces the switch into the boot loader environment (ROMMON):

As previously demonstrated with Labbing With a Cisco Catalyst 2950 switch, I'm going to start off this lab with running the same factory reset commands starting with a flash initialization on the Cisco Catalyst 3550.
