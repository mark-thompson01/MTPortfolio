The following outlines 10 Essential Linux Commands that form a great foundation when starting out with Linux and other Unix-based systems.

Linux is everywhere, over 70% of the world's servers run on Linux. Linux runs on servers, networking equipment, IoT devices, embedded systems, mobile phones, etc. Linux is all over the place!

For anyone working with Linux, having a strong command-line foundation is essential and beneficial depending on what circumstances are encountered. 1. pwd - Print Working Directory

What it does: Shows where you are in the filesystem.

/home/mt directory as the current working directory.



2. ls - List Files and Directories

What it does: Lists the contents of the current directory.

The ls -l does a long listing of the given directory contents to display them vertically on the terminal output.



3. cd - Change Directory

What it does: Moves between directories.

This example navigates from the /home/mt directory to the /var/log directory in the file system hierarchy.



4. mkdir - Make Directory

What it does: Creates new folders.



5. touch - Create an Empty File

What it does: Quickly makes an empty file or updates its timestamp.



6. cat - Concatenate and Display File Contents

What it does: Displays the contents of files.

This example displays - concatenates the os-release information of the Linux distribution, residing in /etc/os-release.



7. grep - Search Inside Files

What it does: Searches for patterns in text files.



8. chmod - Change File Permissions

What it does: Adjusts file read/write/execute permissions.

+x is the argument that instructs chmod to add (+) executable (x) permissions to the specified file.



9. chown - Change File Ownership

What it does: Changes the user and group ownership of a file.



10. man - Manual Pages

What it does: Display help documentation for any command.




Mastering these commands builds a foundation for navigating and working with a Linux system. They're generally the first set of tools you start out with when navigating the file system of a Linux or Unix-like system.
