Download & Install PowerShell on Ubuntu

This guide walks through installing PowerShell on Ubuntu 24.04 using the Microsoft repository, importing the signing key, and verifying a successful installation.

PowerShell is fully cross-platform thanks to its .NET Core foundation. Installing PowerShell on Ubuntu allows Linux users to leverage familiar cmdlets, automation, and scripting capabilities directly within the terminal environment.

The following steps walk through updating your system, installing prerequisites, importing the Microsoft GPG key, enabling the Microsoft repository, and installing the powershell package.

Step 1 — Update and Refresh Package Index

Before installing new software, it’s best practice to update Ubuntu’s package index. This ensures the package manager has the latest metadata and prevents version conflicts when adding new repositories.

Step 2 — Verify System Information (Optional)

Some installations depend on release codename or architecture. Checking system details provides clarity and ensures commands are compatible with your Ubuntu version.

Step 3 — Prepare the System

PowerShell installation requires certain prerequisite packages such as curl, apt-transport-https, and certificate utilities. These enable secure communication with Microsoft's package servers.

Step 4 — Install Prerequisite Packages

This command installs all necessary tools for securely downloading and validating Microsoft’s signing key. These prerequisites ensure apt can trust and process packages from external repositories.

Step 5 — Confirm Prerequisites were Installed

Ubuntu confirms the installation of required packages. If all dependencies are satisfied, you’re ready to continue importing the Microsoft signing key.

Step 6 — Import Microsoft’s GPG Key

Microsoft signs all packages with a GPG key. Importing this key allows Ubuntu to verify that PowerShell packages come from a trusted and authenticated source.

Step 7 — Add the Microsoft Repository

After the signing key is installed, the Microsoft repository must be added to Ubuntu's sources list. This step enables apt to fetch PowerShell directly from Microsoft’s servers.

Step 8 — Update Package Index Again

Once the new repository is added, apt needs to be refreshed to recognize available PowerShell packages from Microsoft’s repo.

Step 9 — Install PowerShell

The PowerShell package can now be installed normally using apt. This downloads the latest stable version for Ubuntu and registers the pwsh interpreter.

Step 10 — Verify Installation

Running pwsh --version confirms a successful installation. You can now launch PowerShell on Ubuntu by typing pwsh into the terminal.
