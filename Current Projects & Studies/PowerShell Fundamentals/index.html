PowerShell Fundamentals

PowerShell is a powerful command-line shell and scripting language built on .NET. Designed for system administrators and power users, it automates repetitive tasks, manages system configurations, and interacts with APIs, registry, and file systems. It is object-based (unlike traditional text-based shells), meaning it passes objects between commands (not just raw text), which gives it unique flexibility.

Core Cmdlets and Concepts

Cmdlet = Command + Let (lightweight commands)

  • Get-Command – Lists available cmdlets
  • Get-Help – Displays help info about cmdlets
  • Get-Service – Shows running services
  • Set-ExecutionPolicy – Changes script execution policy
Get-Help Get-Service -Full

Variables and Data Types

$Name = "Mark"
$Age = 29
$Services = Get-Service

Variables start with $
Data types: string, int, array, boolean, object

Loops and Conditional Statements

# If-Else
if ($Age -gt 18) {
    Write-Output "Adult"
} else {
    Write-Output "Minor"
}

# ForEach Loop
$services = Get-Service
foreach ($service in $services) {
    Write-Output $service.DisplayName
}

Functions and Scripting

function Greet-User {
    param ($Name)
    Write-Output "Welcome, $Name!"
}

Greet-User -Name "Mark"

Create reusable code blocks
Scripts end with .ps1 extension

Useful Tools and Debugging
