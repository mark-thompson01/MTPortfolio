Hyper-V Administrative Tasks II

This page continues where the first Administrative Tasks walkthrough left off — going over other VM tasks, Hyper-V host settings, Quick Create workflows, and virtual disk editing operations within Hyper-V Manager.

1. VM Tasks

General virtual machine tasks commonly performed inside Hyper-V Manager.

Opening the VM Task Menu

Right-click on a given VM to access options such as: Connect to VM, Access VM Settings, Turn Off, Shut Down, Reset (Reboot), and more.

Managing VM Options

Managing VM options such as Start, Stop, Checkpoints, and Settings.

2. Hyper-V Settings

Host-level configuration screens for managing global Hyper-V platform behavior.

General Overview Panel

Hyper-V Settings — general overview panel for host configuration.

Default VM Configuration and Storage Locations

Adjusting default VM configuration and storage locations.

NUMA Spanning Support

Configuring NUMA spanning support for multi-processor environments.

Enhanced Session Mode Policy

Configuring Enhanced Session Mode Policy for interactive VM access.

Keyboard & Mouse Console Control

Setting keyboard pass-through options and mouse release key combinations for VM console control.

Enhanced Session Mode & Reset Check Boxes

Enhanced Session Mode user-level setting, and the Reset Check Boxes option — used to restore the default behavior of confirmation messages and wizard pages.

3. Quick Create

The step-by-step workflow for creating a virtual machine using the Quick Create wizard.

Launching the Wizard & Selecting an Image

In the upper right-hand side under Actions, click Quick Create... then select a base operating system image.

Downloading the Image

Beginning the Quick Create image download for the Windows 11 dev environment.

Extracting & Creating the Disk

Extracting the disk from the image archive and creating the virtual hard drive.

Configuring the VM

Setting up the OS installation environment, allocating processor cores and RAM, and assigning the VM to a vSwitch.

Moving the VM to a New Location

To relocate the VM, right-click and select Move, then follow the Move Wizard.

4. Edit Virtual Disk (Edit vDisk)

A walkthrough of modifying and managing virtual hard disk files (.vhdx) using the Edit Disk wizard.

Launching the Edit Disk Wizard

Right-click on the Hyper-V hostname in the upper-left and select Edit Disk..., or click it in the Hyper-V Manager Actions pane on the right.

Locating the Virtual Hard Disk

Choosing the Edit Action

Select the desired action — Expand, Shrink, or Convert. In this example, Convert is selected.

Choose Disk Format & Type

Converting from VHDX to VHD, then selecting Fixed size as the disk type.

Completing the Conversion

Converting the virtual hard disk and completing the Edit Virtual Hard Disk Wizard.
