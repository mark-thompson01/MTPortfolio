Initial System Setup III

User account creation and configuration for the extended Hyper-V lab environment.

Step 1 Windows Settings - Accounts

Opening Windows Settings and navigating to the Accounts section. This is where user accounts for the Hyper-V lab system are managed. The current account "LAB-CLIENT" is a Local Account with Administrator privileges.



Step 2 Other Users - Add Account

Accessing the "Other users" section to add a new user account. The existing user "Mark Thompson" (Administrator) is already configured. Clicking "Add account" to create a new local user account for the lab environment.



Step 3 Create User Account - Initial Form

The "Create a user for this PC" dialog opens. This form allows creation of a local user account with a username and password. Important notes about child accounts and password security are displayed. Ready to enter the new username.



Step 4 Create User Account - Enter Details

Entering "Lab-Client User" as the username for the new account. This will be a standard user account (non-administrator) that can be used for testing and regular operations in the Hyper-V lab environment. Password fields are ready to be filled in.



Step 5 User Account Created Successfully

The new user account "Lab-Client User" has been successfully created and now appears in the "Other users" list as a Local account. The account is ready to be used for lab operations and can be further configured with specific permissions as needed.
