PHASE 3 – Users & Groups (RBAC Properly)

Implement role-based access control using the AGDLP model with security groups and test users.

Step 9 – Create Security Groups (AGDLP Model)

Under each department:

Create:

Global Security Groups

  • IT_Users
  • HR_Users
  • Sales_Users

Then create:

Domain Local Groups (for permissions later)

  • DL_IT_Share_RW
  • DL_HR_Share_RW
  • DL_Sales_Share_RW

Add:

  • Users → Global Groups
  • Global Groups → Domain Local Groups (later when file server built)

Step 10 – Create Test Users

Create 2–3 per department:

Example:

  • it.admin
  • hr.user1
  • sales.user1

Add to appropriate Global Group.
