PHASE 7 – Hardening (Professional Touch)

Apply enterprise-level security hardening measures to protect the domain environment.

Step 20 – Disable SMBv1

PowerShell on servers:

Disable-WindowsOptionalFeature -Online -FeatureName SMB1Protocol

Step 21 – Enable Advanced Audit Policy

GPO:

Computer Configuration → Policies → Windows Settings → Security Settings → Advanced Audit Policy

Enable:

  • Logon/Logoff
  • Account Management
  • Object Access

Step 22 – Document Everything

Create documentation covering:

  • Network diagram
  • IP addressing scheme
  • OU structure
  • Group Policy settings
  • NTFS permissions model
  • Security hardening measures applied
