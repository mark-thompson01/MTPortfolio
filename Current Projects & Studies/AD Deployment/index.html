Active Directory Deployment – SOHO Enterprise Simulation

This lab focuses on setting up and administrating an Active Directory SOHO Enterprise Simulation. Designing and deploying a secure, scalable Active Directory environment that would serve as a standard implementation for a 25-user small business.

Outlined throughout these lab modules are industry best practices for setting up an AD implementation, things that should be considered before getting started, default configurations to know about early on, AD security hardening & best practices for overall AD security, best practices for AD administration, and a more in-depth look at managing AD with the use of PowerShell.

Let's dive in!

Scope:

  • Domain Controller deployment
  • Organizational Unit design
  • Group Policy implementation
  • File Server with NTFS security
  • Role-based access control
  • Baseline hardening

Technologies Used:

  • Windows Server 2022
  • Windows 11
  • Hyper-V
  • PowerShell
  • Group Policy Management
