Windows PowerShell Troubleshooting Scripts

A collection of PowerShell scripts developed to automate system troubleshooting across network connectivity, disk health, event logs, services, user profiles, and Windows updates.

View this project on GitHub

1. Network Connectivity Troubleshooting Script

This script tests network adapters, gateway reachability, and DNS resolution while generating a full connectivity report.

2. Disk Space and Drive Health Check

Checks available disk space and identifies drives with less than 15% free capacity. Exports low-space results to CSV.

3. Windows Service Status Troubleshooter

Verifies essential Windows services like BITS, Spooler, WinRM, and W32Time. Automatically restarts any that are stopped and logs the results.

4. Event Log Error Scanner

Scans for critical System event errors from the last 24 hours and exports them to CSV for easier analysis.

5. User Profile or Login Issue Troubleshooter

Detects corrupted user profiles and `.bak` registry entries that may cause login problems. Results are exported to a CSV for inspection.

6. Windows Update and Compliance Check

Lists installed updates, identifies failed ones (Event ID 20), triggers a scan, and logs results for update compliance.

Summary

Together, these scripts streamline troubleshooting and reduce manual effort across a Windows environment. Each script includes exportable logs, allowing IT professionals to review historical data or automate routine checks across multiple systems.
